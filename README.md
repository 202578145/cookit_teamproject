# 🍳 Cookit - 냉장고 속 재료로 찾는 나만의 레시피

**Cookit**은 "냉장고에 남은 재료로 뭘 해 먹을까?"라는 고민을 해결해 주는 웹 기반 레시피 추천 서비스입니다.
사용자가 가진 식재료(예: 계란, 양파)를 검색하면 관련된 요리를 찾아주고, 매일 새로운 메뉴를 추천해 줍니다.

> **University Team Project**
> * 📅 **개발 기간:** 2025.12.15 (약 6시간)
> * 🛠 **사용 기술:** HTML5, CSS3, JavaScript (Vanilla)
> * 🧑‍🎓 정윤서 (202578145)

---

## 🚀 주요 기능 (Key Features)

### 1. 🔍 스마트 레시피 검색
- 냉장고에 있는 재료(예: egg, onion)를 검색창에 입력하면 관련 레시피를 찾아줍니다.
- 검색 결과 페이지에서 요리의 사진과 간단한 설명을 확인할 수 있습니다.

### 2. 🎲 오늘의 메뉴 추천 (Today's Pick)
- 매번 무엇을 먹을지 고민될 때, '새로고침(🔄)' 버튼을 누르면 랜덤으로 메뉴를 추천해 줍니다.
- 메인 화면에서 추천 메뉴의 사진과 바로가기 링크를 제공합니다.

### 3. ✨ 재미있는 인터랙티브 효과
사용자의 즐거운 경험을 위해 다양한 시각/청각 효과를 구현했습니다.
- **🎵 BGM Player:** 상단 스피커 아이콘(🔇/🔊)을 클릭하면 배경음악을 켜고 끌 수 있습니다.
- **🌧️ Food Rain Effect:** 'Food Rain' 버튼을 누르면 하늘에서 맛있는 식재료 이모지들이 눈처럼 내립니다.

### 4. 📱 반응형 및 확장형 레이아웃
- **메인 화면:** 인기 메뉴 3가지를 카드 형태로 깔끔하게 보여줍니다.
- **전체 메뉴 보기:** `>` 버튼을 클릭하면 `All Recipes` 페이지로 이동하여, 9가지의 모든 메뉴를 리스트 형태로 한눈에 볼 수 있습니다.

## ⚠️ 유의사항 및 환경 (Notice & Environment)

웹사이트 이용 시 원활한 경험을 위해 아래 내용을 참고해 주세요.

* **🔊 브라우저 정책 (Auto-play Policy):** 최신 브라우저의 보안 정책에 따라, 배경음악(BGM)은 사용자가 페이지와 상호작용(클릭 등)을 한 이후부터 재생이 가능할 수 있습니다. 소리가 나지 않는다면 화면을 한 번 클릭해 주세요.
* **🖥️ 최적화 환경 (Optimization):** 본 서비스는 **PC Chrome(크롬) 및 Edge(엣지)** 브라우저, **FHD 해상도(1920x1080)**에 최적화되어 있습니다. 모바일 환경에서는 일부 레이아웃이나 애니메이션 효과가 의도와 다르게 보일 수 있습니다.
* **🥦 이모지 호환성 (Emoji Compatibility):** 'Food Rain' 기능 등에 사용된 이모지(🥦, 🥩 등)는 사용자의 운영체제(Windows, Mac, iOS) 폰트에 따라 디자인이 다르게 렌더링될 수 있습니다.
